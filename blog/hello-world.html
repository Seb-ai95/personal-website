Hello, world: building this site

Jun 18, 2026

This is the first post on my new site. I wanted somewhere to keep my work and my writing in one place, without reaching for a heavy framework. So this is just HTML and CSS.

The design system

Everything you see is built on a small editorial design system called Mosaic Press: cream paper, ink-black panels, and a high-voltage violet accent. The components live in a single system.css file, so every page just links it and uses the mp-* classes.

Depth comes from color blocking, not glassy shadows.

Why no framework?

  • It loads instantly and hosts anywhere, including GitHub Pages.
  • There's no build step to babysit.
  • Every post is a plain file I can copy from a template.

Writing a new post

To add an article I copy post-template.html, fill in the title and body, and add a card on the blog index. That's the whole workflow:

cp blog/post-template.html blog/my-new-post.html
# edit the title and prose, then link it from blog/index.html
